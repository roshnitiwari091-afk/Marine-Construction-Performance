# Marine Construction Project Performance 

## 📌 Project Overview

This project analyzes marine construction project data to identify key factors affecting project delays, cost overruns, and resource utilization.

## 🛠 Tools Used

* SQL (SQLite Online)
* Python (Pandas, Matplotlib)
* Power BI (Dashboard Visualization)

## 📊 Key KPIs

* Average Project Delay (Days)
* Total Budget vs Actual Cost
* Cost Overrun
* Resource Utilization

## 📈 Dashboard Insights

* Offshore projects show the highest cost overruns
* Projects with higher equipment usage tend to exceed budget
* Weather delays significantly impact project completion time
* Certain locations consistently show higher delays

## 📂 Project Structure

* Dataset (CSV)
* SQL Queries
* Power BI Dashboard Screenshot

## 🚀 Outcome

Built an end-to-end analytics solution to monitor project performance and support data-driven decision-making.
